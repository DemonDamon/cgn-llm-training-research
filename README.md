# CGN领域大模型训练与评估研究

**CGN LLM Training & Evaluation Research**

本仓库系统性收集了关于**多源异构数据融合、数据工程、大模型训练、评估方法论及智能体应用**的技术资料和研究文献。

## 🎯 项目目标

1. 研究多源异构数据资产治理方案，建设高质量数据集
2. 构建垂直领域大模型，提升专业任务性能
3. 开发智能体应用，实现业务流程自动化
